# Focus Guard

Focus Guard is a Flutter + Android accessibility app that helps reduce doomscrolling and short-form video addiction.

## What It Does

- Detects Shorts and Reels style content in supported apps and browsers
- Tracks phone usage as waste, productive, or neutral
- Lets you define rules per app or browser
- Blocks short-form feeds after your limit is reached

## Current Features

- YouTube Shorts detection
- Instagram and Facebook Reels detection
- X/Twitter short video and status-page heuristics
- Browser support for Chrome, Firefox, Edge, Brave, Samsung Internet, Opera, and Vivaldi
- Rule-based app categorization
- On-device usage dashboard

## Tech

- Flutter for the app UI
- Android AccessibilityService for detection and blocking
- SharedPreferences for local rule and stats storage
